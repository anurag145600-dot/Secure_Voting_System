# Secure Voting System

A Secure Voting System developed using Flask, SQLite and Paillier Homomorphic Encryption.

## Features

- User Registration
- Login Authentication
- OTP Verification
- Secure Voting
- Admin Dashboard
- Result Analysis
- Vote Visualization

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js
- Paillier Cryptosystem
